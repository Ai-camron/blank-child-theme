# -BLANK- ‚Äî Anything But Blank

**By Papi & Pup LLC**

A motion-led apparel brand site hosted on **Hostinger.com**, built with **WordPress**, **WooCommerce**, **Elementor**, and the **Astra** theme. Originally prototyped in **React**, **Vite**, **Tailwind CSS**, **framer-motion**, **lucide-react**, and **React Router**, the design and animations have been adapted for a seamless, premium e-commerce experience.

## üé® Brand Design Language

From the rebrand deck:

* **Core Elements:**

  * Clean silhouettes with thoughtful proportions.
  * One or two bold colors per collection (e.g., Cobalt Blue, Persimmon Orange, Orchid Lavender).
  * Geometric graphics, misaligned blocks, intentional asymmetry.
  * Typography: refined serif paired with monospaced sans for balance.
  * Iconography: signals, glyphs, windows, error marks.
